### D3.js: Basic SVG Chart Setup (JavaScript)

Source: https://d3js.org/getting-started

This snippet demonstrates how to set up a basic SVG container for a D3.js chart. It includes defining dimensions, margins, and creating scales for the x and y axes. Dependencies include the D3.js library.

```javascript
{
  // Declare the chart dimensions and margins.
  const width = 640;
  const height = 400;
  const marginTop = 20;
  const marginRight = 20;
  const marginBottom = 30;
  const marginLeft = 40;

  // Declare the x (horizontal position) scale.
  const x = d3.scaleUtc()
      .domain([new Date("2023-01-01"), new Date("2024-01-01")])
      .range([marginLeft, width - marginRight]);

  // Declare the y (vertical position) scale.
  const y = d3.scaleLinear()
      .domain([0, 100])
      .range([height - marginBottom, marginTop]);

  // Create the SVG container.
  const svg = d3.create("svg")
      .attr("width", width)
      .attr("height", height);

  // Add the x-axis.
  svg.append("g")
      .attr("transform", `translate(0,${height - marginBottom})`)
      .call(d3.axisBottom(x));

  // Add the y-axis.
  svg.append("g")
      .attr("transform", `translate(${marginLeft},0)`)
      .call(d3.axisLeft(y));

  // Return the SVG element.
  return svg.node();
}
```

### Declarative Line Plot in Svelte with D3

Source: https://d3js.org/getting-started

Implements a line plot in Svelte using D3.js modules that do not manipulate the DOM (shape, scale). It leverages Svelte's reactive statements (`$:`) for declarative rendering of the visualization.

```svelte
<script>
  import * as d3 from 'd3';

  export let data;
  export let width = 640;
  export let height = 400;
  export let marginTop = 20;
  export let marginRight = 20;
  export let marginBottom = 20;
  export let marginLeft = 20;

  $: x = d3.scaleLinear([0, data.length - 1], [marginLeft, width - marginRight]);
  $: y = d3.scaleLinear(d3.extent(data), [height - marginBottom, marginTop]);
  $: line = d3.line((d, i) => x(i), y);
</script>
<svg width={width} height={height}>
  <path fill="none" stroke="currentColor" stroke-width="1.5" d={line(data)} />
  <g fill="white" stroke="currentColor" stroke-width="1.5">
    {#each data as d, i}
      <circle key={i} cx={x(i)} cy={y(d)} r="2.5" />
    {/each}
  </g>
</svg>
```

### Line Plot with Axes in React using D3 and useEffect

Source: https://d3js.org/getting-started

Integrates D3.js modules that manipulate the DOM (selection, transition, axis) within a React component. It uses `useRef` and `useEffect` to manage D3's DOM interactions, rendering dynamic axes.

```jsx
import * as d3 from "d3";
import {useRef, useEffect} from "react";

export default function LinePlot({
  data,
  width = 640,
  height = 400,
  marginTop = 20,
  marginRight = 20,
  marginBottom = 30,
  marginLeft = 40
}) {
  const gx = useRef();
  const gy = useRef();
  const x = d3.scaleLinear([0, data.length - 1], [marginLeft, width - marginRight]);
  const y = d3.scaleLinear(d3.extent(data), [height - marginBottom, marginTop]);
  const line = d3.line((d, i) => x(i), y);
  useEffect(() => void d3.select(gx.current).call(d3.axisBottom(x)), [gx, x]);
  useEffect(() => void d3.select(gy.current).call(d3.axisLeft(y)), [gy, y]);
  return (
    <svg width={width} height={height}>
      <g ref={gx} transform={`translate(0,${height - marginBottom})`} />
      <g ref={gy} transform={`translate(${marginLeft},0)`} />
      <path fill="none" stroke="currentColor" strokeWidth="1.5" d={line(data)} />
      <g fill="white" stroke="currentColor" strokeWidth="1.5">
        {data.map((d, i) => (<circle key={i} cx={x(i)} cy={y(d)} r="2.5" />))}
      </g>
    </svg>
  );
}
```

### D3.js Transition Control Flow

Source: https://d3js.org/d3-transition/control-flow

Explains the lifecycle of a D3.js transition, from creation to completion, including scheduling, starting, running, and ending phases. It also details how to configure transitions and handle events like 'interrupt' and 'start'.

```APIDOC
## D3.js Transition Lifecycle

### Description

This section describes the various stages a D3.js transition goes through, from its creation and configuration to its execution and eventual end or interruption. It covers how methods like `delay`, `duration`, `attr`, and `style` are used, and how events like `start` and `interrupt` are dispatched.

### Configuration and Scheduling

- Transitions can be configured immediately after creation using methods like `_selection_.transition`, `_transition_.delay`, `_transition_.duration`, `_transition_.attr`, and `_transition_.style`.
- Methods specifying target values are evaluated synchronously.
- Methods like `_transition_.attrTween` and `_transition_.styleTween` require deferred evaluation until the transition starts.
- Once scheduled, the transition's `delay` and `start` event listeners cannot be modified without throwing an error.

### Transition Start and Interruption

- Upon starting, a transition interrupts any active transition of the same name on the same element, dispatching an `interrupt` event.
- It cancels pending transitions of the same name.
- The `start` event is dispatched, marking the last point at which the transition can be modified.

### Transition Execution

- Tweens are initialized immediately after starting.
- Tweens are invoked for each frame with an eased `t`-value from 0 to 1.
- Tweens are invoked in the order they were registered within each frame.

### Transition End

- Upon completion, tweens are invoked a final time with `t = 1`.
- An `end` event is dispatched.
- The transition is deleted from the element, and its configuration is destroyed.
```

### Chained Transitions with d3.active() in JavaScript

```javascript

    .delay((d, i) => i * 50)
    .on("start", function repeat() {

            .style("fill", "red")
          .transition()
            .style("fill", "green")
          .transition()
            .style("fill", "blue")
          .transition()
            .on("start", repeat);
      });
```

### D3.js Drag Event Listener Example

Source: https://d3js.org/d3-drag

This JavaScript code snippet demonstrates how to handle drag events in D3.js. It defines listeners for the 'drag' and 'end' events within a 'start' event handler. The 'dragged' function updates the element's position and the 'ended' function removes a 'dragging' class.

```javascript
function started(event) {
  const circle = d3.select(this).classed("dragging", true);
  const dragged = (event, d) => circle.raise().attr("cx", d.x = event.x).attr("cy", d.y = event.y);
  const ended = () => circle.classed("dragging", false);
  event.on("drag", dragged).on("end", ended);
}
```
