### Pandas Series head() Method Example

Source: https://pandas.pydata.org/docs/development/contributing_docstring

Demonstrates the usage of the `head()` method for the Pandas Series object. It shows how to retrieve the first elements of a Series and how to specify the number of elements to return using the `n` parameter.

```python
class Series:

    def head(self, n=5):
        """
        Return the first elements of the Series.

        This function is mainly useful to preview the values of the
        Series without displaying all of it.

        Parameters
        ----------
        n : int
            Number of values to return.

        Return
        ------
        pandas.Series
            Subset of the original series with the n first values.

        See Also
        --------
        tail : Return the last n elements of the Series.

        Examples
        --------
        >>> ser = pd.Series(['Ant', 'Bear', 'Cow', 'Dog', 'Falcon',
...                'Lion', 'Monkey', 'Rabbit', 'Zebra'])
        >>> ser.head()
        0   Ant
        1   Bear
        2   Cow
        3   Dog
        4   Falcon
        dtype: object

        With the ``n`` parameter, we can change the number of returned rows:

        >>> ser.head(n=3)
        0   Ant
        1   Bear
        2   Cow
        dtype: object
        """
        return self.iloc[:n]

```

### Pandas: Read and write compressed pickle files using read_pickle and to_pickle

Source: https://pandas.pydata.org/docs/whatsnew/v0.20.0

Demonstrates the new capability of `read_pickle` and `DataFrame/Series.to_pickle` to handle compressed pickle files. Compression can be specified explicitly or inferred from the file extension, supporting formats like gzip and bz2.

```python
import pandas as pd
import numpy as np

df = pd.DataFrame({'A': np.random.randn(1000),
                   'B': 'foo',
                   'C': pd.date_range('20130101', periods=1000, freq='s')})

# Explicitly specify compression (gzip)
df.to_pickle("data.pkl.compress", compression="gzip")
rt_explicit = pd.read_pickle("data.pkl.compress", compression="gzip")
print("Explicit gzip compression read:\n", rt_explicit.head())

# Infer compression from file extension (.gz)
df.to_pickle("data.pkl.gz")
rt_infer_gz = pd.read_pickle("data.pkl.gz")
print("Inferred .gz compression read:\n", rt_infer_gz.head())

# Infer compression from file extension (.bz2) for a Series
s1 = df["A"]
s1.to_pickle("s1.pkl.bz2")
rt_infer_bz2 = pd.read_pickle("s1.pkl.bz2")
print("Inferred .bz2 compression read (Series):\n", rt_infer_bz2.head())
```

### Querying and chunking data with Pandas and PyTables

Source: https://pandas.pydata.org/docs/user_guide/io

Shows how to append data to an HDF5 store, define a helper function for chunking, and then select data based on specific criteria using `select_as_coordinates` and custom chunking. This allows for processing subsets of data efficiently.

```python
store.append("dfeq", dfeq, data_columns=["number"])

def chunks(l, n):
    return [l[i: i + n] for i in range(0, len(l), n)]

evens = [2, 4, 6, 8, 10]
coordinates = store.select_as_coordinates("dfeq", "number=evens")

for c in chunks(coordinates, 2):
    print(store.select("dfeq", where=c))

```
